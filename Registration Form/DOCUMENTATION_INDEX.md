## 💡 **Common Questions**

### Q: How do I start?
A: Open `index.html` in your browser, then click "Login" and use demo account:
- Email: `user@example.com`
- Password: `Password123!`

### Q: Where's the code?
A: All JavaScript code is in the `js/` folder:
- `validation.js` - All validation logic
- `registration.js` - Registration form handler
- `login.js` - Login form handler

### Q: How do I test everything?
A: See `TESTING.md` - it has 50+ test cases you can run

### Q: Can I use this for production?
A: Yes, but you should:
1. Use proper bcrypt password hashing
2. Implement server-side validation
3. Use HTTPS always
4. Consult security documentation in README.md

### Q: What browsers work?
A: Chrome, Firefox, Safari, Edge, and mobile browsers (see README.md)

### Q: How does data get saved?
A: All data is stored in browser's localStorage (see README.md for details)
