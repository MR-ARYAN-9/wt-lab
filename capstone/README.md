# Student Project Management System (SPMS)

This is a lightweight capstone-ready web application built with PHP, MySQL, HTML, CSS, and JavaScript.

Requirements:
- PHP 7.4+ (with mysqli)
- MySQL / MariaDB
- Apache (XAMPP/WAMP)

Setup:
1. Place the project folder in your web server root (e.g., `C:/xampp/htdocs/capstone`).
2. Create the database and tables using `sql/schema.sql` in phpMyAdmin or CLI.
3. Update `config.php` DB credentials if needed.
4. Ensure `uploads/` folder is writable by the web server.
5. Open `index.php` in browser.
